# How To Use 使用方法

1. Type `div` in the file of supported languages above.  在所支持语言文件中键入 `div`。
2. Select `div (Divider)` and press `Tab`.  选择 `div (Divider)` 后按下 `Tab`。
3. Divider will automatically be added to the file.  分隔符将自动加入文件中。
